## ⚠️ Signup Not Working?

**Issue:** Email confirmation is enabled in Supabase

**Quick Fix:**
1. Go to: https://supabase.com/dashboard
2. Select project: `iekygvfianzgklwpgiqr`
3. Navigate to: **Authentication → Settings → Email Auth**
4. **Disable:** "Enable email confirmations"
5. Save changes
6. Signup will now work immediately!

**Alternative:** Create accounts via Supabase Dashboard:
- Go to: **Authentication → Users**
- Click: **"Add user" → "Create new user"**
- Check: **"Auto Confirm User"**
- Enter credentials above
