### Explorer Technical Implementation

The explorer is positioned **fixed on the side of the viewport** (left or right edge, slightly inward) and his vertical position, animation state, and environment react to scroll progress.

```javascript
// Explorer scroll tracking
const scrollProgress = window.scrollY / (document.body.scrollHeight - window.innerHeight);

// Map scroll progress to explorer Y position
const explorerY = scrollProgress * (window.innerHeight * 0.7) + (window.innerHeight * 0.15);

// Map scroll progress to animation state
if (scrollProgress < 0.15) setExplorerState('walking-fast');
else if (scrollProgress < 0.30) setExplorerState('looking-up');
else if (scrollProgress < 0.50) setExplorerState('walking-slow');
else if (scrollProgress < 0.70) setExplorerState('exploring');
else if (scrollProgress < 0.90) setExplorerState('resting');
else setExplorerState('arrived');
```

- Dark charcoal version (on light backgrounds)
- Smooth crossfade when switching animation states

**Explorer animation states (CSS/SVG animation):**
- `walking`: legs cycle, slight body bob
- `looking-up`: static pose, slight tilt of head
- `pointing`: one arm extends toward screen
- `sitting`: sitting pose, gentle breathing bob

### Button Hover Interaction

On hover, the following happen simultaneously:

1. **Cursor shine sweep:** The `::before` shine element sweeps across the button from left to right — `0.6s` duration, cinematic easing
2. **Glow intensifies:** `box-shadow` expands — warm gold outer glow: `0 0 40px rgba(201, 169, 110, 0.2)`
3. **Subtle lift:** `translateY(-2px)` — button rises slightly
4. **Border brightens:** border opacity increases from `0.4` → `0.7`
5. **Magnetic cursor:** Button slightly attracts cursor when within `80px` radius

```javascript
// Magnetic button interaction
button.addEventListener('mousemove', (e) => {
  const rect = button.getBoundingClientRect();
  const centerX = rect.left + rect.width / 2;
  const centerY = rect.top + rect.height / 2;
  
  const deltaX = (e.clientX - centerX) * 0.18;
  const deltaY = (e.clientY - centerY) * 0.18;
  
  button.style.transform = `translate(${deltaX}px, ${deltaY}px) translateY(-2px)`;
});

button.addEventListener('mouseleave', () => {
  button.style.transform = 'translate(0, 0)';
});
```
