## CURSOR SYSTEM — PREMIUM MAGNIFYING GLASS (FULL SPEC)

Apply across the entire homepage. This was specified in v3 and must be fully implemented.

### Philosophy

The cursor is an **emotional exploration tool** — not a gimmick.
Every time the user moves over text or imagery, they are **discovering something deeper**.
It must feel luxurious, smooth, artistic, and alive.

Must NOT feel: gimmicky, cartoonish, laggy, or like a tech demo.

---

### Cursor Structure — 3 Layers

**Layer 1 — Inner lens:**
```css
.cursor-lens {
  width: 48px;
  height: 48px;
  border-radius: 50%;
  border: 1px solid rgba(201, 169, 110, 0.5);
  background: rgba(255, 248, 235, 0.04);
  backdrop-filter: blur(0.5px);
  box-shadow:
    inset 0 0 12px rgba(255, 248, 235, 0.06),
    0 0 20px rgba(201, 169, 110, 0.08),
    0 4px 20px rgba(0, 0, 0, 0.15);
  position: fixed;
  pointer-events: none;
  z-index: 9999;
}
```

**Layer 2 — Outer ring:**
```css
.cursor-ring {
  width: 72px;
  height: 72px;
  border-radius: 50%;
  border: 1px solid rgba(201, 169, 110, 0.15);
  position: fixed;
  pointer-events: none;
  z-index: 9998;
}
```

**Layer 3 — Magnifying glass handle:**
- Short diagonal line extending from bottom-right of lens
- Length: `12px` · Thickness: `1.5px` · `rotate(45deg)`
- Color: `rgba(201, 169, 110, 0.45)` · Rounded end cap
- This detail makes it unmistakably a magnifying glass — without being cartoonish

---

### Movement — Lerp Smoothing

```javascript
let mouse = { x: 0, y: 0 };
let lens  = { x: 0, y: 0 };
let ring  = { x: 0, y: 0 };

const LENS_LERP = 0.12;  // inner — slightly faster
const RING_LERP = 0.07;  // outer — more lag = more luxury feel

document.addEventListener('mousemove', (e) => {
  mouse.x = e.clientX;
  mouse.y = e.clientY;
});

function animateCursor() {
  lens.x += (mouse.x - lens.x) * LENS_LERP;
  lens.y += (mouse.y - lens.y) * LENS_LERP;
  ring.x += (mouse.x - ring.x) * RING_LERP;
  ring.y += (mouse.y - ring.y) * RING_LERP;

  cursorLens.style.transform   = `translate(${lens.x - 24}px, ${lens.y - 24}px)`;
  cursorRing.style.transform   = `translate(${ring.x - 36}px, ${ring.y - 36}px)`;
  cursorHandle.style.transform = `translate(${lens.x + 16}px, ${lens.y + 16}px) rotate(45deg)`;

  requestAnimationFrame(animateCursor);
}
animateCursor();
```

The cursor must feel: **fluid, inertial, alive — never snappy or mechanical.**

---

### Text Magnification Effect

When the cursor moves over ANY text element — taglines, quotes, destination names, captions, body copy — the text **subtly magnifies inside the lens area**.

- Zoom level inside lens: `1.35x – 1.45x`
- Transition in: `0.3s` ease
- Transition out: `0.4s` ease (slightly slower exit = premium feel)
- Serif details become more visible and refined inside the lens
- A very soft warm glow traces letterforms
- Text feels weighted, dimensional, important

**Key cinematic lines react most expressively:**
```
Hovering "choose yourself"
→ Letters expand softly within lens
→ Serif curves become beautifully detailed
→ Warm gold glow traces the letterforms

Hovering destination names
→ Name sharpens and grows inside lens
→ Subtle atmospheric warmth behind letters
→ Feels like focusing on something far away
```

### Cursor Rules

- `cursor: none` on `body` — custom cursor replaces ALL native cursors
- All cursor animations on `requestAnimationFrame` only — never `setTimeout`
- `will-change: transform` on lens element — GPU composited
- Target: `60fps` minimum at all times
- **Completely disabled on touch/mobile devices** — no fallback shown
- The website must remain perfectly readable at all times — if in doubt, reduce the effect
- Luxury is restraint — never exaggerate the magnification
