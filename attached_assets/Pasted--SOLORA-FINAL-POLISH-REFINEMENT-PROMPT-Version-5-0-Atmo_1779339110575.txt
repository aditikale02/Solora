## FIX 2 — MAGNIFYING GLASS CURSOR — TRUE TEXT INTERACTION

### Problem
The cursor is currently a styled visual element only.
It does not actually magnify or interact with text underneath it.
The emotional interaction system is missing entirely.

### Fix — Real Text Magnification on Hover

---

### Approach — CSS Scale + Clip-path Method (Recommended)

This approach scales text elements near the cursor position and clips the effect to the lens boundary — creating a true magnification illusion without heavy WebGL.

```javascript
// True text magnification system
const lens = document.querySelector('.cursor-lens');
const MAGNIFY_RADIUS = 48;      // px — matches lens size
const MAGNIFY_SCALE = 1.38;     // zoom level — subtle, luxury
const LERP = 0.12;

let mouseX = 0, mouseY = 0;
let lensX = 0, lensY = 0;

// Track all text elements
const textTargets = document.querySelectorAll(
  'h1, h2, h3, h4, p, .quote, .destination-name, .stat-number, .tagline, .memory-caption'
);

document.addEventListener('mousemove', (e) => {
  mouseX = e.clientX;
  mouseY = e.clientY;
});

function updateMagnification() {
  // Smooth lens position
  lensX += (mouseX - lensX) * LERP;
  lensY += (mouseY - lensY) * LERP;

  textTargets.forEach(el => {
    const rect = el.getBoundingClientRect();
    const elCenterX = rect.left + rect.width / 2;
    const elCenterY = rect.top + rect.height / 2;

    // Distance from lens center to element center
    const dist = Math.hypot(lensX - elCenterX, lensY - elCenterY);

    // Influence radius — larger than lens for smooth falloff
    const influenceRadius = MAGNIFY_RADIUS * 2.5;

    if (dist < influenceRadius) {
      // Proximity factor: 1 = directly under lens, 0 = edge of influence
      const proximity = 1 - (dist / influenceRadius);
      const scale = 1 + (MAGNIFY_SCALE - 1) * proximity * proximity;

      el.style.transform = `scale(${scale})`;
      el.style.transformOrigin = `${lensX - rect.left}px ${lensY - rect.top}px`;
      el.style.transition = 'transform 0.15s ease, filter 0.15s ease';

      // Sharpness increase
      const sharpness = proximity * 0.3;
      el.style.filter = `contrast(${1 + sharpness * 0.15}) brightness(${1 + sharpness * 0.05})`;

      // Warm glow on text near lens
      el.style.textShadow = `0 0 ${proximity * 30}px rgba(201, 169, 110, ${proximity * 0.25})`;

    } else {
      // Reset elements outside influence
      el.style.transform = 'scale(1)';
      el.style.filter = 'none';
      el.style.textShadow = '';
    }
  });

  requestAnimationFrame(updateMagnification);
}

updateMagnification();
```

---

### Magnification Behavior Per Text Type

| Text Element | Magnification | Glow | Notes |
|---|---|---|---|
| Hero tagline | `1.35x` | Warm gold soft | Most expressive — the flagship interaction |
| Section headlines | `1.3x` | Subtle warm | Elegant and refined |
| Quote lines | `1.4x` | Strongest glow | These are emotional — react most |
| Destination names | `1.35x` | Warm atmosphere | Feel like focusing on a far place |
| Stat numbers | `1.3x` | Clean sharp | Sharpness increase more than glow |
| Memory captions | `1.25x` | Barely perceptible | Intimate and quiet |
| Body copy / small text | `1.2x` | None | Legibility always priority |

---

### Magnification Rules — Luxury Restraint

- Maximum zoom: `1.45x` — NEVER exceed this
- The effect must feel like **gentle optical focus** — not aggressive zoom
- On mobile/touch: completely disabled
- On elements with `position: fixed` (navbar): disabled
- The website must remain perfectly readable at all times
- If text becomes harder to read due to scaling — reduce the effect

> *Think: editorial fashion website. Not science magnifier.*

## GLOBAL PERFORMANCE RULES

All of the above animations must maintain:

- **60fps minimum** — test on mid-range devices
- All animations: `transform` and `opacity` only — no layout thrashing
- Use `will-change: transform` only on actively animating elements (remove after animation ends)
- Particles: use `canvas` element for large particle systems — NOT individual DOM elements
- Environmental motion: `CSS animations` — lighter than JS
- Magnification system: `requestAnimationFrame` loop — never `setInterval`
- Prefers-reduced-motion: respect this — wrap all decorative animations:

```css
@media (prefers-reduced-motion: reduce) {
  .grass-layer,
  .cloud-layer,
  .ember,
  .floating-particle,
  .explorer-breathe {
    animation: none;
  }
}
```
