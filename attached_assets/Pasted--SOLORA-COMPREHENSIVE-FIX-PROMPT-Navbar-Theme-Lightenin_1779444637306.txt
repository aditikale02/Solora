## FIX 1 — NAVBAR BUTTONS & SMOOTH SCROLL NAVIGATION

### Problem
Navbar buttons don't respond when clicked. Navigation is completely broken.

### Solution

Make every navbar link scroll smoothly to its corresponding section.

**Step 1 — Add IDs to all sections**

Each section must have a unique ID:

**Step 2 — Add click handlers to navbar buttons**

```javascript
// Smooth scroll function
function smoothScroll(targetId) {
  const target = document.getElementById(targetId);
  if (!target) return;
  
  target.scrollIntoView({
    behavior: 'smooth',
    block: 'start'
  });
}

// Attach to navbar buttons
document.addEventListener('DOMContentLoaded', () => {
  const navLinks = {
    'nav-logo': 'hero',
    'nav-explore': 'destinations',
    'nav-stories': 'stories',
    'nav-community': 'community',
    'nav-cta': 'selector'
  };

  Object.entries(navLinks).forEach(([buttonId, sectionId]) => {
    const btn = document.getElementById(buttonId);
    if (btn) {
      btn.addEventListener('click', (e) => {
        e.preventDefault();
        smoothScroll(sectionId);
      });
    }
  });
});
```

**Step 3 — Mobile hamburger menu (if exists)**

If you have a mobile hamburger menu:

// Close menu after click

mobileMenuItems.forEach(item => {
  item.addEventListener('click', () => {
    const menu = document.querySelector('.mobile-menu');
    menu.classList.remove('open'); // Close menu
    smoothScroll(item.getAttribute('href').replace('#', '')); // Scroll

**Step 4 — Active state highlighting (optional but nice)**

Show which section user is currently viewing:

```javascript
function updateActiveNavLink() {
  const sections = document.querySelectorAll('section');
  const navLinks = document.querySelectorAll('[data-nav-link]');
  
  sections.forEach(section => {
    const { top, bottom } = section.getBoundingClientRect();
    if (top < window.innerHeight / 2 && bottom > 0) {
      navLinks.forEach(link => link.classList.remove('active'));
      const activeLink = document.querySelector(`[data-nav-link="${section.id}"]`);
      if (activeLink) activeLink.classList.add('active');
    }
  });
}

window.addEventListener('scroll', updateActiveNavLink);
```

### JavaScript — Duration Calculator

When user selects dates, calculate trip duration:

```javascript
const startDateInput = document.getElementById('start-date');
const endDateInput = document.getElementById('end-date');
const durationValue = document.getElementById('duration-value');

function calculateDuration() {
  if (!startDateInput.value || !endDateInput.value) {
    durationValue.textContent = '—';
    return;
  }

  const start = new Date(startDateInput.value);
  const end = new Date(endDateInput.value);
  const days = Math.ceil((end - start) / (1000 * 60 * 60 * 24));

  if (days < 0) {
    durationValue.textContent = 'Invalid dates';
    return;
  }

  durationValue.textContent = `${days} day${days !== 1 ? 's' : ''}`;
}

startDateInput.addEventListener('change', calculateDuration);
endDateInput.addEventListener('change', calculateDuration);
```
